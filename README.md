# 탄수화물원과 단백질원의 조합이 포만감에 미치는 영향
다이어터의 궁금증: 어떤 식품을 통해 탄수화물과 단백질을 섭취했을 때 가장 포만감이 높을까요? <br>
우리는 해답을 얻기 위해 직접 실험을 설계했습니다! 
## 방법
$2^{2}$ 설계로 요인의 수가 2개이고, 각 요인마다 수준수가 2개인 요인실험을 진행했습니다.
